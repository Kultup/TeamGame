Технічне завдання (TZ) — Інтерактивна гра "Дискусійні картки"
Мета проекту — створити веб-додаток для груп, де гравці обирають випадкові картки (або крутять колесо), за якими приховані завдання або питання для обговорення. Дизайн має бути преміальним, динамічним та з "WOW"-ефектом.

Основний функціонал та наповнення
Категорії карток (на основі ТЗ "Гра в команду: масштабнування"): Гра буде розділена на 7 основних тематичних блоків:

Комунікація (питання про обмін інформацією та зворотний зв'язок).
Прийняття рішень (процеси та відповідальність).
Ролі та обов'язки (хто за що відповідає).
Бізнес-процеси (оптимізація та ефективність).
Мотивація (внутрішні та зовнішні стимули).
Масштабування (ріст команди та бізнесу).
Стратегування (цілі та візія на майбутнє).
Спеціальні картки:

Перерва на творчість: Творчі завдання для "перемикання" уваги.
Зміна курсу: Несподівані повороти в ході гри.
Ігрове поле:

Вибір категорії перед початком або випадковий вибір.
Анімований 3D-переворот картки.
Лічильник відкритих карток.
Дизайн та естетика
IMPORTANT

Дизайн має виглядати сучасно та дорого (Premium UI).

Стиль: Glassmorphism (ефект матового скла), неоморфізм або насичений Dark Mode з градієнтами.
Анімації: Плавні переходи, мікро-взаємодії при наведенні курсору, частинки (particles) або "конфеті" при успішній дії.
Типографіка: Сучасні шрифти (наприклад, Inter або Outfit) для максимальної читабельності.
Технологічний стек
Frontend: React + Vite (забезпечує швидке завантаження та плавність інтерфейсу).
Styling: Vanilla CSS або CSS Modules (для повного контролю над складними анімаціями).
Animations: CSS 3D Transforms або Framer Motion.
План верифікації
Тестування взаємодії: Перевірка коректності 3D-перевороту картки на різних пристроях.
Адаптивність: Коректне відображення на мобільних телефонах і планшетах.
Логіка: Перевірка того, що завдання не повторюються занадто часто або випадково перемішуються.
