# План розвитку проекту: Team Building Game 🚀

Цей документ визначає пріоритетні напрямки розвитку гри для перетворення її на повноцінну платформу.

## 📋 Пріоритетні функції

### 1. Архівація контенту
*   Можливість приховати запитання або цілі категорії в адмін-панелі, не видаляючи їх назавжди.
*   Функція "Відновити" для повернення архівних питань у гру.

### 2. Система голосування
*   Інтерактивне голосування гравців за найкращу або найцікавішу відповідь.
*   Збір "зірочок" або балів гравцями протягом сесії.
*   Фінальний екран з результатами (MVP сесії).

### 3. Командні режими
*   Спеціальні набори питань, адаптовані під стадії розвитку команди за Такменом:
    *   **Формування (Forming)** — акцент на знайомстві та безпеці.
    *   **Штормінг (Storming)** — акцент на вирішенні конфліктів та цінностях.
    *   **Нормування (Norming)** — акцент на правилах та спільних цілях.

### 4. Реальний Backend (MongoDB)
*   Міграція з мок-драйвера (`db.js`) на повноцінну базу даних **MongoDB**.
*   Забезпечення надійного довгострокового збереження питань, категорій та результатів ігор.
*   Підготовка API для багатокористувацької гри.

### 5. Мультимовність
*   Підтримка англійської та інших іноземних мов.
*   Перемикач мов в інтерфейсі гри та адмін-панелі.
*   Можливість додавати переклади для кожного питання.
